# Pet Shop

a web application to buy and sell pets

## Getting Started

- Clone this repository: `git clone https://github.com/Benziza/allom3alem.git`.
- Run Server Tomcat 9 .
- Run MySql database .
- Import file **allom3alem.sql** in your database .
- Run the project from eclipse or intellij.
